# Expense Tracker

Expense Tracker is a web application built using HTML, CSS, JavaScript, and a Java backend with Spring Boot (MVC) for managing monthly budgets and tracking expenses.

## Features
- Add, edit, and delete expenses
- Track monthly budgets
- Track total expense per month
- Categorize expenses (Food, Travel, Entertainment, Self, etc.)
- Mood-based tracking (for future advancements in ML)

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Java with Spring Boot (MVC)
- Database: MySQL

## Future Improvements

- Add monthly savings summary table
- Improve UI design
- Add user authentication
- Integrating with ML to predict and advice user about his expenses and budgeting
